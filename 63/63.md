# ユーザ登録API（課題63）

Node.js + Express を用いて、
ユーザ名とパスワードを受け取り、パスワードをハッシュ化して
データベースに登録する API を実装する。

---


## セットアップ
```
npm init -y
npm install express sqlite3 bcrypt
```
---

## 起動方法
```
node server.js
```
起動後、http://localhost:3000 で待ち受ける。

---

## ユーザ登録API

### エンドポイント
```
POST /register
```
### リクエスト例
```
curl -X POST http://localhost:3000/register \
  -H "Content-Type: application/json" \
  -d '{"username":"user1","password":"pass"}'
```

## データベース

起動時に users.db が自動生成される。
