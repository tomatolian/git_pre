
12. curlコマンドでGETリクエストを送る
curlコマンドを使って https://httpbin.org/get にGETリクエストを送り、返ってきたレスポンスを確認せよ。  


curl は Client URL の略で、URLを指定してHTTP(S)などの通信を行うコマンドラインツール。
`curl URL`でGETリクエストを送ることができます

```
curl https://httpbin.org/get  
> {
  "args": {}, 
  "headers": {
    "Accept": "*/*", 
    "Host": "httpbin.org", 
    "User-Agent": "curl/8.7.1", 
    "X-Amzn-Trace-Id": "Root=1-69002db5-41d88b4301cd44c810a08c2b"
  }, 
  "origin": "133.5.123.96", 
  "url": "https://httpbin.org/get"
}
```

"args": {} はクエリパラメータが空であることを示す

"headers" には送信時のHTTPヘッダが含まれており、User-Agent や Host が確認できる。

"origin" はアクセス元IPアドレス。

"url" は実際にアクセスしたURL。
